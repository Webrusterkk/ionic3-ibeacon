Welcome to Lock

This a demostration of BLE, Socket.io with Ionic. Here we starter project comes with simple tabs-based layout for apps that are going to primarily use a Tabbed UI.

Functioning

Mobile Application listen to the ibeacon packets from the laptop contiuously. As, soon as I move from my seat with my mobile, as some point it will exit from the broadcasting region of laptop. Immediatley it will send a scoket request over the network to the node.js based application running on my ubuntu laptop and locks the laptop. Similarly, as I come back in the broadcasting range it will send another socket request to unlock the laptop.
